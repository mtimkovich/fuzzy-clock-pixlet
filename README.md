# tidbyt-fuzzy-clock

Display the time in a groovy, human-readable way.

## Quick start
1. Install [`pixlet`](https://github.com/tidbyt/pixlet)
2. Run `pixlet serve fuzzy_clock.star`
3. Go to [http://localhost:8080](http://localhost:8080)
